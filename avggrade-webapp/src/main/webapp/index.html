About

  1. This application calculates the weighted average grade of students

  2. The student list gets sorted by their weighted average grade

  3. If the weighted average grade is equal then students are sorted by names

  4. The grades are read in from XML file

  5. Upload button on our demo form below enables you to select XML file from hard drive

  6. Demo requires XML file to contain a list of students with their courses together with the number of credit points and grades like in our example
